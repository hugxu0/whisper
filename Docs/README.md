# Whisper documentation

This directory holds the product behavior brief, frozen protocol boundary,
architecture decisions, build policy, and verification notes for the
clean-slate rewrite.

Fixtures live in the repository-level `Fixtures/` directory so they can be
consumed by both pure domain tests and future client tests.

Architecture decisions that introduce an external dependency are recorded as
ADRs, beginning with `ADR_001_SOCKET_IO.md`.
